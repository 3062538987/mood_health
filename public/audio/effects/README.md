# 解压游戏音效文件

请在此目录下放置以下音效文件：

- `woodenFish.mp3` - 木鱼敲击声
- `paddleHit.mp3` - 挡板接球声
- `brickBreak.mp3` - 砖块破裂声
- `gameOver.mp3` - 游戏结束声
- `tetrisRotate.mp3` - 俄罗斯方块旋转声
- `tetrisClear.mp3` - 俄罗斯方块消除行声
- `tetrisDrop.mp3` - 俄罗斯方块下落声

## 音效要求

- 格式：MP3
- 大小：尽量控制在500KB以内
- 时长：1-2秒为宜
- 音量：适中，不要过大

## 替代方案

如果没有实际音效文件，系统会使用静音处理，不会影响游戏功能。
