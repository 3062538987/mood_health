# 音频文件说明

本目录用于存放音频放松模块所需的音频文件。

## 所需文件

请在本目录中放置以下音频文件：

1. `rain.mp3` - 雨声
2. `ocean.mp3` - 海浪声
3. `fire.mp3` - 篝火声

## 音频来源

您可以从以下网站获取免费的自然声音音频：
- [Freesound](https://freesound.org/)
- [YouTube Audio Library](https://www.youtube.com/audiolibrary/music)
- [Free Music Archive](https://freemusicarchive.org/)

## 注意事项

- 请确保音频文件格式为MP3
- 建议音频文件长度为5-10分钟，以便循环播放
- 音频文件大小建议不超过10MB，以确保加载速度
